# NEXUS — League of Legends Companion App

A personal League of Legends companion application built for private use.

## Features

- **Champion Select Integration** — Auto-detects champion select via LCU API. Displays optimal rune pages, item builds, summoner spells and skill orders for your selected champion
- **Rune Auto-Import** — Automatically imports recommended rune pages directly into the League client with one click
- **Live Build Data** — Aggregates recent ranked match data from Diamond+ players using the Riot Match v5 API to calculate highest win rate builds per champion per rank tier
- **Champion Lookup** — Search any champion with rank-filtered builds (Iron through Diamond+), confidence ratings, and role detection for flex picks
- **Counterpick Suggestions** — Real-time counterpick recommendations as enemies lock in during draft
- **Ban Suggester** — Dangerous champion combo detection with threat ratings
- **Match History** — Detailed post-game breakdown with role-aware AI performance analysis
- **Objective Timers** — In-game Baron, Dragon, Herald and tower timers with audio alerts
- **Patch Notes** — Direct links to latest Riot patch notes with unread notifications

## Technical Stack

- Node.js local server (localhost:3000)
- League Client Update (LCU) API for live client integration
- Riot Games Match v5, Summoner v4, League v4, Account v1 APIs
- Data Dragon for static assets
